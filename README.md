# HUE

**HUE**는 “나의 감정을 색으로 기록한다”는 컨셉의 **감정 다이어리 캘린더 앱**  
사용자는 날짜마다 오늘의 감정을 하나의 색과 짧은 문장으로 남기고,  
색으로 물든 캘린더를 통해 한 달의 감정 흐름을 직관적으로 확인 가능

---

## 프로젝트 소개

### 기획 배경

- **Ionic 프레임워크**를 활용한 다이어리/캘린더 앱을 직접 구현하기 위해 시작한 프로젝트
- 기존 다이어리 앱과의 차별점을 위해,  
  날짜에 텍스트만 남기는 것이 아니라 **감정을 색으로 표현**하는 방식을 도입

### 기획 목적

- 특정 날짜를 기록할 때, 그날의 기분을 **색(배경색/글자색)**으로 표현하고 간단한 텍스트를 함께 저장할 수 있도록 구현
- 감정의 변화를 한눈에 볼 수 있는 **색 기반 감정 캘린더**를 만드는 것이 목표

### 기대 효과

- 달력이 다양한 색으로 채워지면서 일반적인 다이어리 앱보다 **시각적으로 풍부한 경험** 제공
- 색상과 짧은 메모를 함께 남겨,
  “그날 어떤 기분이었는지, 무슨 일이 있었는지”를 나중에 쉽게 되돌아볼 수 있도록 구현

## 주요 기능

### 로그인 / 회원가입

- 앱 사용을 위해 계정 기반 로그인 사용
- 계정이 없는 경우:
  - 아이디 / 비밀번호 / 휴대폰 번호를 입력해 회원가입
- 로그인 성공 시 메인 캘린더 화면으로 이동

### 캘린더 뷰

- Ionic의 `<ion-datetime>` 컴포넌트를 활용한 달력 UI
- 날짜별로 **배경색 / 글자색**을 설정해 감정 상태를 색으로 표현
- 날짜를 선택하면 해당 날짜의 색상과 내용을 확인 가능

### 날짜·색상·내용 추가/수정

- 날짜, 배경색, 글자색, 내용을 입력해 감정 기록 추가
- 같은 날짜가 이미 존재하는 경우:
  - 새로 입력한 값으로 **덮어쓰기(업데이트)** 처리
- 짧은 메모(예: 최대 20자)를 함께 저장하여 간단한 일기 역할 수행

### 계정 관리

- 현재 계정에서 입력한 모든 기록(날짜·색·내용)을 한 번에 삭제
- 로그아웃 기능 제공

## 기술 스택
### Framework

- Ionic (Angular 기반 하이브리드 앱 프레임워크)

### Frontend

- TypeScript
- HTML / CSS
- Ionic UI 컴포넌트 (ion-datetime, ion-input, ion-button 등)

### Backend

- PHP (데이터 입출력용 API)
- MySQL (날짜, 색상, 내용 저장)

### 통신

- Angular HttpClient를 이용한 REST API 호출
